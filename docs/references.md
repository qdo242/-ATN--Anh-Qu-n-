# Tài liệu tham khảo kỹ thuật

## 1. Tiêu chuẩn mật mã học
- **NIST Special Publication 800-38D:** Recommendation for Block Cipher Modes of Operation: Galois/Counter Mode (GCM) and GMAC.
- **FIPS PUB 197:** Advanced Encryption Standard (AES).
- **RFC 2104:** HMAC: Keyed-Hashing for Message Authentication.
- **RFC 6234:** US Secure Hash Algorithms (SHA and SHA-based HMAC and HKDF).

## 2. Giao thức và Kiến trúc
- **RFC 7539:** ChaCha20 and Poly1305 for IETF Protocols (Dùng để so sánh hiệu năng với AES-GCM).
- **ISO/IEC 29192-3:** Information technology - Security techniques - Lightweight cryptography - Part 3: Block ciphers.

## 3. Thư viện triển khai
- **PyCryptodome Documentation:** Thư viện mật mã học cho ngôn ngữ Python.
- **Mbed TLS Documentation:** Thư viện mật mã học cho thiết bị nhúng và vi điều khiển.
